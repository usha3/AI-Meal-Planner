# AI Meal Planner

AI-powered Android application that helps users plan meals, discover recipes, and generate grocery lists.

## Features

- User authentication with Firebase
- AI meal recommendations
- Recipe search and filtering
- Grocery list generation
- Favorites management
- Meal calendar planner
- Room Database storage
- API integration

## Tech Stack

- Java
- Android Studio
- Firebase
- Room Database
- Retrofit
- REST APIs
- AI / Gemini API
